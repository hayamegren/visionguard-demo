# VisionGuard – SecureID AI

VisionGuard is an adaptive login risk engine that analyzes user behavior
and determines access decisions (SAFE / SUSPICIOUS / HIGH RISK).

### Features
- Risk-based authentication
- Step-up verification via Nafath (biometric)
- Explainable security decisions
- Hackathon demo (frontend + python logic)

### Tech Stack
- Python (Isolation Forest – backend logic)
- HTML / CSS / JavaScript (frontend)
- GitHub Pages (deployment)
